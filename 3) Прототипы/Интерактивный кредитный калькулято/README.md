# Демо-проект: Интерактивный кредитный калькулятор
Этот проект создан для демонстрации навыков проектирования сложных интерактивных интерфейсов и работы с анимацией в Figma. 
> **Цель проекта:** Разработать интуитивно понятный и визуально привлекательный инструмент, который позволяет пользователям мгновенно оценивать параметры кредита, экспериментировать с условиями и видеть итоги расчёта в динамике.

# Ключевые элементы интерфейса и логика

**Элементы:** Поля для ввода/слайдеры «Сумма кредита», «Срок кредита», «Ставка», выбор «Типа платежей», кнопка «Рассчитать».

**Логика и анимация:** Предполагается, что изменение любого параметра (например, сдвиг слайдера срока) может запускать мгновенный перерасчёт. Анимация может визуально «перетекать» старые значения в новые в блоке результатов. П
ереключение между типами платежей — ключевая интерактивная функция, полностью меняющая логику расчёта.
