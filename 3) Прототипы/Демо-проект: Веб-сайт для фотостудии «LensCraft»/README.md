# Демо-проект: Веб-сайт для фотостудии «LensCraft»
Этот проект создан для демонстрации навыков прототипирования, проектирования UX/UI и анимации интерфейсов. 
Поскольку показ реальных рабочих проектов ограничен соглашениями о конфиденциальности (NDA), этот кейс наглядно показывает мой подход и уровень владения инструментами.
> Цель проекта: Создать полнофункциональный прототип современного сайта, который отражает ключевые компетенции в визуализации требований и проработке пользовательского опыта.

# Ключевые страницы и особенности
**1. Главная страница**

**Что сделано**: Разработан визуально выразительный верхний блок с четким заголовком. Использованы анимации для привлечения внимания.

**2. Интерактивная галерея работ**

**Что сделано**: Реализована сетка работ с категориями. Элементы портфолио интерактивны — реагируют на наведение, есть плавные переходы для просмотра деталей.

**3. Многошаговая форма бронирования / заявки**

**Что сделано**: Форма разбита на логические шаги или группы полей. Продуманы состояния полей (активное, с ошибкой, заполненное) для улучшения UX.

**4. Адаптивное меню и навигация**

**Что сделано**: Для мобильной версии предусмотрено «гамбургер»-меню с плавной анимацией открытия. На десктопе — классическая горизонтальная панель.
