# Документация
В этой папке собраны примеры документации, которые я создавала в качестве аналитика в рабочих проектах. В компании использовались упрощенные виды постановок, поэтому форма ГОСТ 34 не применялась под каждую доработку. Есть примеры ТЗ, оформленного по ГОСТ 34, но из-за секретности данных, я не могу поместить его в данный репозиторий.
# Постановки
По причине секретности данных, некоторые рисунки и атрибуты скрыты из постановок.
# Инструкции
Инструкции публиковались в специальном ПО - Help + manual. В данном репозитории лежат артефакты, разрабатываемые перед публикацией в Help + manual.
